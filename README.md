## MERN URL Shortener
### Summary 
This App is URL Shortener v0.0.1

The technologies stack consists of:
* MongoDB
* Express.js
* React.js (Hooks)
* Node.js

How to run the app:
* install node modules for the client and server
`npm install`
`npm run client:install`

* start the app
`npm run dev`

* go to http://localhost:5000 in your browser
